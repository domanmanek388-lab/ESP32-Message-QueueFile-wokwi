# Message-QueueFile-wokwi
Project Komunikasi antar Task Menggunakan Queue (Wokwi)

Project IoT menggunakan ESP32 dan sensor Potentiometer
yang disimulasikan menggunakan Wokwi.

## Fitur
- Membaca durasi kedipan led
- Menampilkan data analog Potentiometer per putaran
- pushbuton untuk mengontrol kedipan led

## Teknologi
- ESP32
- Arduino IDE
- Wokwi Simulator
